# What is FQDN?

**FQDN** stands for **Fully Qualified Domain Name**.

## Definition

An FQDN is the complete domain name for a specific computer, host, or service on the internet or within a private network. It uniquely identifies a device or resource by specifying its exact location in the Domain Name System (DNS) hierarchy.

## Structure of an FQDN

An FQDN consists of several parts:

- **Hostname:** The name of the specific machine or service (e.g., `mail`, `www`, `server1`).
- **Domain name:** The domain under which the host resides (e.g., `example.com`).
- **Top-Level Domain (TLD):** The highest level in the DNS hierarchy (e.g., `.com`, `.org`, `.net`).

An FQDN always ends with a dot (`.`) representing the root of the DNS hierarchy, though this trailing dot is often omitted in everyday use.

## Example
www.example.com.
- `www` is the hostname.
- `example` is the domain.
- `com` is the top-level domain.
- The trailing dot represents the DNS root.

## Importance of FQDN

- Provides an unambiguous way to identify a host on the internet or network.
- Used in DNS resolution to map domain names to IP addresses.
- Essential for configuring network services, SSL certificates, email servers, etc.
