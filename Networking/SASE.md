## Secure Access Service Edge
It is made of 4 technologies on high level
- Network
    - Secure web gateway (SWG)
    - Cloud access security broker (CASB)
    - Zero trust network access (ZTNA)
- Security
    - Software defined wide area network (SDWAN)
